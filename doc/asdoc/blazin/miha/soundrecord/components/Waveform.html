draw()method
public function draw(bytes:ByteArray, stretchSignal:Boolean = true, precision:int = 10):void

Draws the waveform given the recorded sound. Stretches the waveform to stage.stageWidth. If stretchSignal is set to true, the signal height is stretched to a maximum of the waveform height. The precision parameter defines the number of lines to draw per pixel. The higher the value, the more precise the waveform but the slower the performance as more lines get drawn per pixel.

Parameters

bytes:ByteArray — Bytes recorded by the microphone
 
stretchSignal:Boolean (default = true) — Whether to stretch the signal height or not
 
precision:int (default = 10) — Number of lines to draw per pixel.
